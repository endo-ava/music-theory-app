# 画面仕様詳細: About (Philosophy)

[<< 画面設計書（全体）に戻る](../0002.screenDesign.md)

## 1. 画面概要

- **画面ID**: `SCR-003`
- **画面名**: About
- **概要**: Harmonic Orbitの設計思想と世界観（Philosophy）を伝えるための画面。「使い方の説明」ではなく、「なぜこの視点が必要なのか」という問いに対する答えを提示する。

## 2. コンテンツ構成

### 2.1 Philosophy (導入)

- **メッセージ**: Harmonic Orbitは、音楽理論を「知識」から「体験」へと昇華させるための装置であること。

### 2.2 Why Circle? (なぜ五度圏なのか)

- **内容**: 音楽の循環性、オクターブの同一性、五度圏の永遠の回帰について。直線的な鍵盤では見えない「重力関係」が円環構造にあることを説明。

### 2.3 Why Set Theory? (なぜ集合論なのか)

- **内容**: 既存の「文法」に縛られない、音そのものの物理現象・数理的秩序へのアプローチ。ピッチクラス・セット理論を採用することで、普遍的な構造分析が可能になること。

### 2.4 What is Atlas? (Atlasとは何か)

- **内容**: 単なる用語集ではなく、概念間の関係性を可視化した「知の地図」であること。断片的な知識を体系的な理解へ導く羅針盤としての役割。

## 3. デザイン方針

- **タイポグラフィ**: 可読性が高く、かつ洗練されたフォントを使用し、メッセージの重みを伝える。
- **レイアウト**: 余白を十分に取ったシングルカラムレイアウトで、読むことに集中できる環境を作る。
- **ビジュアル**: 抽象的な幾何学模様や、アプリ内のビジュアル要素を背景や挿絵として控えめに使用し、世界観を統一する。
